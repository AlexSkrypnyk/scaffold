# Auto assign PR

The GitHub Action named "Auto Author Assign" is designed to automatically assign
the author to pull requests in a GitHub repository. It gets triggered whenever a
pull request is either opened or reopened. The Action requires write permissions
to pull requests in order to carry out its operations.

The main action utilized within this job is [`toshimaru/auto-author-assign`](https://github.com/toshimaru/auto-author-assign).
